capabilities

## Structure

### **Tempaltes:**


- **Argo Applications**

- **ArgoConfig**

    This contains the project definitons  for the argo project
    This  contains the argo defintions for the namespaces kube-addons

- **RBAC**

    This contains the rbac configs for the kube-addons ns
